# Neuro-Linguistic Translation Skill

Este skill está diseñado para transformar contenido entre idiomas (principalmente Inglés <-> Español) manteniendo la precisión técnica, el tono de autoridad y la elegancia estética de **INsitu AI**.

## 🎯 Objetivos

- **Precisión Técnica**: Mantener términos como "ROAS", "LLM", "Embedding", "Vector Database" sin traducciones literales incorrectas.
- **Tono Premium**: Utilizar un lenguaje sofisticado, directo y profesional.
- **Adaptación Cultural**: Ajustar modismos y estructuras para que el contenido parezca nativo, no traducido.

## 🛠 Protocolo de Traducción

### 1. Análisis de Contexto

Antes de traducir, identifica:

- **Tipo de Contenido**: (Blog, UI/UX, Documentación Técnica, Ad Copy).
- **Audiencia**: (Media Buyers, Dueños de Agencias, Desarrolladores).

### 2. Reglas de Oro (Glosario Intocable)

No traduzcas los siguientes términos a menos que sea necesario para la fluidez extrema:

- **IA/AI**: Mantener según el contexto del mercado.
- **Frameworks**: Next.js, Tailwind, React.
- **Métricas**: CTR, CPC, CPM, ROAS, LTV.
- **Conceptos de IA**: Fine-tuning, Prompt Engineering, RAG.

### 3. Estructura de Salida

La traducción debe respetar el **formato mandatorio** de INsitu AI:

- **Títulos**: Siempre en Uppercase (si es Blog).
- **Headers H2**: Claros y con punch (White style).
- **Headers H3**: Enfocados en beneficios (Magenta style).

## 📝 Prompts de Activación

### Traducción de Blog Post
>
> "Traduce el siguiente post de [Idioma Origen] a [Idioma Destino]. Utiliza el tono de un Consultor Senior de Marketing. Asegúrate de que los H2 sean potentes y los H3 utilicen un lenguaje persuasivo. Mantén las etiquetas HTML intactas pero traduce el contenido interno."

### Traducción de UI Elements
>
> "Localiza las siguientes cadenas de texto para la interfaz de INsitu AI. Prioriza la brevedad y el uso de verbos de acción fuertes. Evita traducciones literales de Google Translate; usa términos estándar de la industria SaaS."

## 🧪 Verificación de Calidad

1. ¿El texto fluye de forma natural?
2. ¿Se mantiene el énfasis en las palabras clave (Strong tagging)?
3. ¿El tono es aspiracional y tecnológico?
